# Автоматизация тестов для Stellar Burgers. Автотесты для API

[Веб‑приложение Stellar Burgers](https://stellarburgers.education-services.ru/)  

## Подготовка проекта
Создать Maven‑проект и подключить следующие библиотеки:
* JUnit 4 — для написания и запуска тестов
* RestAssured — для тестирования API
* Allure — для создания отчётов по результатам тестирования

## Задание

[Документация API](https://code.s3.yandex.net/qa-automation-engineer/python-full/diploma/Api-Stellar_Burgers.pdf) 

Протестировать эндпоинты API для Stellar Burgers. Тестировать только указанные в задании эндпоинты

### Создание пользователя:
* Создать уникального пользователя
* Создать пользователя, который уже зарегистрирован
* Создать пользователя и не заполнить одно из обязательных полей

### Логин пользователя:
* Вход под существующим пользователем
* Вход с неверным логином и паролем

### Создание заказа:
* С авторизацией
* Без авторизации
* С ингредиентами
* Без ингредиентов
* С неверным хешем ингредиентов

## Отчётность
По результатам выполнения тестов необходимо сформировать отчёт Allure
